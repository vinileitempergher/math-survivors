# Math Survivors 2

A 2D action game built with Godot where players battle mathematical enemies to survive, level up, and discover powerful upgrades.

## Overview

Math Survivors 2 is a roguelike-style game combining action gameplay with math-themed enemies and challenges. Players navigate a dynamically generated world, defeat enemies, collect experience gems, and unlock new abilities to progress further.

## Features

- **Dynamic Combat System**: Attack enemies with various weapons including ice spears, javelins, and tornadoes
- **Enemy Variety**: Battle different enemy types with unique behaviors (Kobolds, Cyclops, Juggernauts, and more)
- **Progression System**: Collect experience gems to level up and unlock powerful upgrades
- **Procedural World**: Randomly generated maps with trees, rocks, and obstacles
- **Audio & Music**: Immersive background music and sound effects for actions and events
- **GUI System**: Intuitive inventory and upgrade selection interface

## Getting Started

### Requirements

- Godot Engine 4.x
- Basic system: Windows, macOS, or Linux

### Installation

1. Clone or download this project
2. Open the project folder in Godot Engine
3. Run the game by pressing `F5` or clicking the Play button

## Game Structure

- **Player/**: Player character and attack mechanics
- **Enemy/**: Enemy types and behaviors
- **World/**: Map generation and boundary systems
- **Objects/**: Collectible items and experience gems
- **Audio/**: Background music and sound effects
- **Textures/**: Game sprites and visual assets
- **Utility/**: UI elements, spawners, and database systems
- **TitleScreen/**: Menu and game entry point

## Controls

- **Movement**: Arrow Keys or WASD
- **Attack**: Mouse Click or Space
- **Select Upgrades**: Click on upgrade options
